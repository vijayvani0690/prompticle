## How to Use This Guide

1. **Choose a combination** from the list below
2. **Copy the prompt** into Bolt.new, Lovable, or v0
3. **Take a screenshot** of the generated result
4. **Save with the filename** provided
5. **Optimize** using Squoosh.app (convert to WebP, 1200px width, 85% quality)

## Naming Convention

Format: `{type}-{style}-{layout}.png`

5. Save with correct filename
6. Move to `/public/previews/{category}/` folder
